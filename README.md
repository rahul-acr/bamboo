# Bamboo

Bamboo is a simple command line tool for creating regular backups

## Description

Bamboo is simple command line tool 


## Installation

Just download this repository and keep the bin directory in  `PATH`


## Usage

```bash
bamboo profile
bamboo device
bamboo backup -auto
```
